# 🚀 Сборка NeetOs через GitHub Actions

## Автоматическая сборка ISO в облаке - БЕЗ Linux на твоём ПК!

---

## 📋 Что нужно (все бесплатно):

1. ✅ Аккаунт на GitHub (https://github.com)
2. ✅ 10 минут времени
3. ✅ Всё! Серьёзно, это всё!

---

## 🎯 Пошаговая инструкция

### ШАГ 1: Создай аккаунт на GitHub

Если его еще нет:
1. Иди на https://github.com/signup
2. Зарегистрируйся (email, пароль, username)
3. Подтверди email

---

### ШАГ 2: Создай новый репозиторий

1. **Зайди на GitHub** и нажми **"+"** → **"New repository"**

2. **Заполни:**
   - Repository name: `neetos`
   - Description: `NeetOs - Custom Linux Distribution`
   - ☑ Public (или Private - не важно)
   - ☑ Add a README file (необязательно)

3. **Нажми** "Create repository"

---

### ШАГ 3: Загрузи файлы проекта

#### Способ A: Через Web-интерфейс (проще)

1. **На странице репозитория** нажми **"Add file"** → **"Upload files"**

2. **Перетащи ВСЮ папку `neetos`** в окно браузера:
   - Все `.md` файлы
   - Все `.sh` файлы
   - Все `.txt` файлы
   - Папки `configs/` и `airootfs/`
   - Папку `.github/` (ВАЖНО!)
   - Файлы `profiledef.sh`, `packages.x86_64`, `pacman.conf`

3. **В поле commit message** напиши: `Initial commit`

4. **Нажми** "Commit changes"

#### Способ B: Через Git (если умеешь)

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/твой-username/neetos.git
git push -u origin main
```

---

### ШАГ 4: Запусти сборку!

1. **Перейди на вкладку "Actions"** в своём репозитории

2. **Если появится запрос** "Enable Actions" - нажми **"I understand, enable them"**

3. **Слева в списке** найди workflow **"Build NeetOs ISO"**

4. **Нажми на него**, затем справа **"Run workflow"**

5. **Выбери** `main` (или `master`) и нажми зелёную кнопку **"Run workflow"**

---

### ШАГ 5: Жди результата

**Процесс сборки:**

```
⏳ Queued...              (0-2 минуты)
🔄 Set up Arch Linux      (2-5 минут)
📦 Install packages       (10-20 минут)
🔨 Build ISO              (10-15 минут)
✅ Upload artifact        (2-5 минут)

Итого: 25-45 минут
```

Ты можешь:
- Закрыть вкладку и вернуться позже
- Или следить за процессом в реальном времени

---

### ШАГ 6: Скачай готовый ISO!

Когда статус станет **✅ зелёным**:

1. **Нажми на название** завершённого workflow

2. **Прокрути вниз** до раздела **"Artifacts"**

3. **Скачай** файл **"neetos-iso"** (это будет ZIP-архив)

4. **Распакуй ZIP** - внутри будет файл **`neetos-YYYY.MM.DD-x86_64.iso`**

5. **Готово!** 🎉

---

## 🎮 Запуск ISO в VirtualBox

Теперь когда у тебя есть ISO:

1. **Создай новую VM:**
   - Тип: Linux → Arch Linux (64-bit)
   - ОЗУ: 4096 MB (или больше)
   - Диск: 20 GB
   - Видео: 128 MB, 3D включено

2. **Подключи ISO:**
   - Настройки → Носители
   - Контроллер IDE → Добавь скачанный `neetos-*.iso`

3. **Запусти VM!**

4. **Логин:** `neet` / `neet`

---

## 🔧 Если хочешь изменить и пересобрать

1. **Внеси изменения** в файлы на GitHub:
   - Измени `packages.x86_64` (добавь/удали пакеты)
   - Измени конфиги в `configs/`
   - И т.д.

2. **Commit changes**

3. **Снова запусти workflow** "Build NeetOs ISO"

4. **Получи новый ISO** с твоими изменениями!

## 📊 Ограничения GitHub Actions

### Бесплатный аккаунт:
- ✅ 2000 минут в месяц (хватит на ~40-50 сборок)
- ✅ Публичные репозитории - неограниченно
- ✅ Файлы до 2GB (наш ISO ~2.5GB, но сжимается)

### Если превысишь лимит:
- Подожди до следующего месяца
- Или создай новый аккаунт
- Или оплати ($0.008/минута)

## ❓ Часто задаваемые вопросы

### Q: Workflow завершился с ошибкой?
A: Смотри логи:
   1. Нажми на failed workflow
   2. Нажми на job "build"
   3. Разверни failed step
   4. Скопируй ошибку и спроси меня

### Q: ISO не собрался?
A: Проверь:
   - Все ли файлы загружены (особенно `.github/workflows/`)
   - Папка `.github` должна начинаться с точки
   - Не забыл ли `profiledef.sh`, `packages.x86_64`

### Q: Можно ли использовать для коммерции?
A: Да, NeetOs под GPL-3.0, делай что хочешь!

### Q: Сколько раз можно собирать?
A: Сколько хочешь! Лимит - 2000 минут/месяц бесплатно

### Q: Artifact исчезает?
A: Да, через 7 дней. Скачай сразу или пересобери.
