ШАГ 1: Иди на GitHub
───────────────────────────────────────
🌐 https://github.com/signup

Если нет аккаунта - зарегистрируйся (2 минуты)
Если есть - залогинься


ШАГ 2: Создай репозиторий
───────────────────────────────────────
1. Нажми "+" в правом верхнем углу
2. Выбери "New repository"
3. Repository name: neetos
4. ☑ Public
5. Кнопка "Create repository" → ГОТОВО!


ШАГ 3: Загрузи все файлы
───────────────────────────────────────
1. Нажми "Add file" → "Upload files"

2. Перетащи ВСЮ папку neetos в окно браузера
   (или выбери все файлы через "choose your files")

3. ВАЖНО! Должны загрузиться:
   ✓ Папка .github (с точкой в начале!)
   ✓ Папка configs
   ✓ Папка airootfs
   ✓ Все .md, .sh, .txt файлы
   ✓ profiledef.sh, packages.x86_64, pacman.conf

4. Внизу в commit message: "Initial commit"

5. Нажми "Commit changes"

⏳ Подожди пока все файлы загрузятся


ШАГ 4: Запусти сборку!
───────────────────────────────────────
1. Вкладка "Actions" (вверху на странице репозитория)

2. Если попросит - нажми "I understand, enable them"

3. Слева выбери "Build NeetOs ISO"

4. Справа кнопка "Run workflow"

5. Выбери "main" → зелёная "Run workflow"

🎉 СБОРКА ПОШЛА!


ШАГ 5: Жди и скачай ISO
───────────────────────────────────────
⏰ Время: 30-40 минут

Можешь закрыть вкладку и вернуться позже!

Когда появится зелёная галочка ✅:
  1. Нажми на workflow
  2. Прокрути вниз до "Artifacts"
  3. Скачай "neetos-iso" (ZIP)
  4. Распакуй → получишь neetos-*.iso

🎮 Запускай в VirtualBox!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❓ Workflow не появился?
   → Проверь загрузилась ли папка .github/workflows/
   → Перезагрузи страницу

❓ Сборка упала с ошибкой?
   → Нажми на failed workflow
   → Разверни failed step
   → Скопируй ошибку и спроси меня

❓ Не могу скачать ISO?
   → Artifact хранится 7 дней
   → Если удалился - пересобери (Run workflow снова)

❓ Artifact слишком большой?
   → GitHub ограничение: 2GB на artifact
   → Наш ISO ~2.5GB, но сжимается до ~1.8GB
   → Должно пройти

❓ Можно приватный репозиторий?
   → Да! Но минуты Actions будут считаться
