## Описание ключевых компонентов

### Скрипты сборки

**build.sh**
- Главный скрипт автоматизации
- Проверяет зависимости
- Создает рабочую директорию
- Копирует конфигурации
- Запускает mkarchiso
- Создает финальный ISO

**check-requirements.sh**
- Проверяет системные требования
- Валидирует наличие пакетов
- Проверяет свободное место
- Тестирует интернет-соединение

**create-structure.sh**
- Создает необходимую структуру директорий
- Копирует конфиги в airootfs
- Устанавливает права на файлы

### Конфигурационные файлы

**profiledef.sh**
- Определяет метаданные ISO
- Устанавливает имя дистрибутива
- Настраивает режимы загрузки
- Определяет права доступа к файлам

**packages.x86_64**
- Список всех пакетов для установки
- Включает базовую систему
- Графическое окружение
- Предустановленный софт
- Драйверы виртуальных машин

**pacman.conf**
- Конфигурация пакетного менеджера
- Включенные репозитории
- Настройки параллельной загрузки

### Файловая система (airootfs)

**airootfs/root/**
- Файлы root-пользователя
- Автоматические скрипты настройки
- Запускаются при первой загрузке

**airootfs/etc/skel/**
- Шаблон домашней директории
- Копируется каждому новому пользователю
- Содержит все персональные настройки

**airootfs/usr/local/bin/**
- Кастомные скрипты NeetOs
- Утилиты системы
- Вспомогательные программы

### Конфигурации приложений

**Hyprland (configs/hyprland/)**
- Оконный менеджер Wayland
- Настройки анимаций, эффектов
- Горячие клавиши
- Правила для окон

**Waybar (configs/waybar/)**
- Панель задач
- Модули мониторинга
- Стили и цвета

**Kitty (configs/kitty/)**
- Эмулятор терминала
- Цветовая схема
- Настройки шрифта

**Rofi (configs/rofi/)**
- Application launcher
- Тема и стили
- Поведение меню

**SDDM (configs/sddm/)**
- Экранный менеджер
- Настройки входа
- Автологин (опционально)

## Процесс сборки

1. **Подготовка**: `check-requirements.sh` проверяет систему
2. **Инициализация**: `build.sh` создает рабочую директорию
3. **Копирование**: Профиль и конфиги копируются в work/
4. **Сборка**: `mkarchiso` собирает ISO из профиля
5. **Результат**: Готовый ISO в директории out/

## Кастомизация

Для изменения любого аспекта системы:

1. Отредактируй соответствующий файл в configs/ или airootfs/
2. Запусти `sudo ./build.sh` для пересборки
3. Протестируй новый ISO в VM

Подробнее: [CUSTOMIZATION.md](CUSTOMIZATION.md)

## Размеры

- **Исходники проекта**: ~5 MB
- **Рабочая директория**: ~15 GB (временная)
- **Финальный ISO**: ~2-3 GB
- **Установленная система**: ~6-8 GB

## Права доступа

Важные исполняемые файлы:
```bash
chmod +x build.sh
chmod +x check-requirements.sh
chmod +x create-structure.sh
chmod +x airootfs/usr/local/bin/*
chmod +x airootfs/root/.automated_script.sh
```

Эти права устанавливаются автоматически скриптом сборки.
