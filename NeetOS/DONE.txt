🎯 ВОТ КАК ЭТО РАБОТАЕТ:
═══════════════════════════════════════════════════════════════════════════

1. Ты загружаешь проект на GitHub
2. GitHub Actions запускает контейнер с Arch Linux
3. В контейнере автоматически:
   ├─ Устанавливается archiso
   ├─ Копируются твои конфигурации
   ├─ Скачиваются все пакеты
   ├─ Собирается ISO-образ
   └─ ISO сохраняется как Artifact
4. Ты скачиваешь готовый ISO!
