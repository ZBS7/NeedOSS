📋 PROJECT OVERVIEW
--------------------------------------------------------------------------------
Name:           NeetOs
Version:        1.0
Base:           Arch Linux (via Archiso)
Architecture:   x86_64
Target:         Virtual Machines (VirtualBox, VMware)
License:        GPL-3.0

Description:
A modern, beautiful, and functional Linux distribution based on Arch Linux,
optimized for virtual machines. Features a cyberpunk-themed Wayland desktop
environment with Hyprland, pre-configured tools, and gaming capabilities.

🎨 FEATURES
--------------------------------------------------------------------------------

DESKTOP ENVIRONMENT:
  • Wayland display server with XWayland compatibility
  • Hyprland compositor with animations and blur effects
  • Waybar status bar with system monitoring
  • Rofi application launcher
  • SDDM login manager
  • Cyberpunk/Cyber-Minimalism theme

PRE-INSTALLED SOFTWARE:
  • Terminal: Kitty + Zsh + Powerlevel10k
  • Browser: Firefox (dark theme)
  • File Manager: Thunar
  • Code Editors: VS Code, Neovim
  • System Monitor: btop, htop
  • Gaming: Steam, Wine, Lutris
  • Development: Git, Node.js, Python
  • Package Management: Pacman, Flatpak

OPTIMIZATIONS:
  • VirtualBox Guest Additions
  • VMware Tools (open-vm-tools)
  • Btrfs filesystem with zstd compression
  • PipeWire audio system
  • NetworkManager for connectivity

  • RAM: 4GB+

  • Packages: archiso, git, base-devel

TARGET VM:
  • Platform: VirtualBox 6.0+ or VMware Workstation 15+
  • RAM: 4GB+ (recommended 8GB)

  • Boot: UEFI and BIOS support

🚀 BUILD PROCESS
--------------------------------------------------------------------------------

QUICK BUILD (3 commands):
  1. sudo pacman -S archiso git base-devel
  2. chmod +x build.sh
  3. sudo ./build.sh

DETAILED BUILD:
  1. Check requirements:    ./check-requirements.sh
  2. Prepare structure:     ./prepare-build.sh
  3. Build ISO:            sudo ./build.sh
  4. Result:               ./out/neetos-YYYY.MM.DD-x86_64.iso

BUILD TIME: 20-40 minutes (depends on internet speed)
ISO SIZE: ~2.5 GB
INSTALLED SIZE: ~6-8 GB

🆘 TROUBLESHOOTING QUICK REFERENCE
--------------------------------------------------------------------------------

COMMON ISSUES:

1. "archiso not found"
   → sudo pacman -S archiso

2. "Permission denied"
   → chmod +x build.sh && sudo ./build.sh

3. "No space left"
   → Free up 20GB+ disk space

4. Black screen after boot
   → Increase VM RAM to 8GB, enable 3D acceleration

5. No internet in VM
   → Check NetworkManager: sudo systemctl start NetworkManager

6. Hyprland won't start
   → Check logs: cat ~/.cache/hyprland/hyprland.log

7. Build fails downloading packages
   → Update mirrors: sudo reflector --latest 20 --save /etc/pacman.d/mirrorlist

See TROUBLESHOOTING.md for complete solutions.
