**NeetOs** — современная, красивая и функциональная операционная система на базе Arch Linux, оптимизированная для работы в виртуальных машинах.

## Архитектурный план

### Базовые компоненты:
- **База**: Arch Linux (через Archiso)
- **Архитектура**: x86_64
- **Загрузчик**: systemd-boot (UEFI) + GRUB (BIOS)
- **Графический сервер**: Wayland + XWayland
- **Оконный менеджер**: Hyprland (с эффектами, анимациями, размытием)
- **Экранный менеджер**: SDDM с кастомной темой NeetOs
- **Панель**: Waybar (кастомизированная)
- **Файловая система**: Btrfs с zstd-сжатием
- **Терминал**: Kitty + Zsh + Powerlevel10k
- **Драйверы VM**: VirtualBox Guest Additions + open-vm-tools

### Предустановленный софт:
- Файловый менеджер: Thunar
- Браузер: Firefox (тёмная тема)
- Редактор кода: VS Code
- Терминальные утилиты: btop, neovim, git
- Гейминг: Steam, Wine, Lutris
- Магазин приложений: Pamac + Flatpak + Snap

### Дизайн:
- Стиль: Cyber-Minimalism / Cyberpunk
- Тема: Тёмная с неоновыми акцентами
- Иконки: Papirus-Dark
- Эффекты: Размытие, закругленные углы, плавные анимации

## Требования для сборки

### Системные требования:
- Хост-система: Arch Linux или Arch-based дистрибутив
- Доступное место: ~20GB
- ОЗУ: 4GB+
- Интернет-соединение

### Установка зависимостей:
```bash
sudo pacman -S archiso git base-devel
```

## Инструкция по сборке

### Шаг 1: Клонирование и подготовка
```bash
# Перейди в директорию с проектом
cd neetos

# Дай права на выполнение скрипта сборки
chmod +x build.sh
```

### Шаг 2: Запуск сборки
```bash
# Запусти автоматическую сборку (требуется sudo)
sudo ./build.sh
```

Скрипт автоматически:
1. Создаст рабочую директорию
2. Скопирует профиль Archiso
3. Настроит список пакетов
4. Добавит кастомные конфигурации
5. Соберёт ISO-образ

### Шаг 3: Результат
После успешной сборки (20-40 минут) ты получишь:
```
./out/neetos-<version>-x86_64.iso
```

## Структура проекта

```
neetos/
├── build.sh                    # Главный скрипт сборки
├── README.md                   # Эта инструкция
├── airootfs/                   # Файловая система для live-системы
│   ├── etc/
│   │   ├── skel/              # Домашняя директория пользователя
│   │   └── systemd/           # Сервисы
│   └── usr/
│       └── local/bin/         # Кастомные скрипты
├── configs/                    # Конфигурационные файлы
│   ├── hyprland/              # Конфиг Hyprland
│   ├── waybar/                # Конфиг Waybar
│   ├── kitty/                 # Конфиг терминала
│   ├── sddm/                  # Тема SDDM
│   └── themes/                # GTK темы и иконки
├── packages.x86_64             # Список пакетов для установки
└── profiledef.sh              # Определение профиля Archiso
```

## Кастомизация

### Изменение списка пакетов:
Отредактируй `packages.x86_64` и добавь/удали нужные пакеты.

### Изменение темы:
Конфиги находятся в `configs/` и `airootfs/etc/skel/.config/`

### Автозапуск приложений:
Добавь `.desktop` файлы в `airootfs/etc/skel/.config/autostart/`
