# Пошаговая инструкция по сборке NeetOs

## Шаг 0: Подготовка хост-системы

### Требования:
- **ОС**: Arch Linux или Arch-based (Manjaro, EndeavourOS)
- **ОЗУ**: 4GB+
- **Диск**: 20GB+ свободного места
- **Интернет**: Стабильное соединение

### Если у тебя Windows:
Используй WSL2 с Arch Linux или запусти Arch в VirtualBox.

### Если у тебя другой Linux:
Лучший вариант - запустить Arch в Docker или VM.

## Шаг 5: СБОРКА ISO

```bash
# Дай права на выполнение
chmod +x build.sh

# ЗАПУСТИ СБОРКУ (требуется sudo)
sudo ./build.sh
```

### Что происходит:

1. **[0-2 мин]** Проверка зависимостей
   ```
   [*] Проверка зависимостей...
   [✓] Все зависимости установлены
   ```

2. **[0-1 мин]** Создание рабочей директории
   ```
   [*] Создание рабочей директории...
   [✓] Рабочая директория создана
   ```

3. **[0-1 мин]** Копирование профиля
   ```
   [*] Копирование базового профиля releng...
   [*] Настройка профиля NeetOs...
   [✓] Конфигурации скопированы
   ```

4. **[20-40 мин]** Сборка ISO
   ```
   [*] Начинается сборка ISO...
   [!] Это может занять 20-40 минут...
   
   [mkarchiso] Загрузка пакетов...
   [mkarchiso] Установка пакетов...
   [mkarchiso] Создание образа squashfs...
   [mkarchiso] Сборка ISO...
   ```

5. **[Финиш]** Завершение
   ```
   ======================================
     ✓ Сборка успешно завершена!
   ======================================
   
   ISO-образ находится в:
     ./out/neetos-2026.06.23-x86_64.iso
   ```

### Время сборки:
- **Быстрый интернет**: 20-25 минут
- **Средний интернет**: 30-35 минут
- **Медленный интернет**: 40+ минут

### Размер загрузки:
- Пакеты: ~2-3 GB

## Шаг 8: Тестирование в VirtualBox

### Создание VM:

1. **Открой VirtualBox**
2. **Создай новую VM:**
   - Имя: `NeetOs`
   - Тип: `Linux`
   - Версия: `Arch Linux (64-bit)`

3. **Настрой оборудование:**
   - ОЗУ: `4096 MB` (или больше)
   - Процессоры: `2` ядра (или больше)

4. **Настрой дисплей:**
   - Видеопамять: `128 MB`
   - ☑ **Включить 3D-ускорение**

5. **Создай виртуальный диск:**
   - Тип: `VDI`
   - Размер: `20 GB`

6. **Подключи ISO:**
   - Настройки → Носители
   - Контроллер IDE → Добавить оптический привод
   - Выбери `neetos-*.iso`

7. **Включи EFI (опционально):**
   - Настройки → Система
   - ☑ Включить EFI

### Запуск:

1. **Запусти VM**
2. **Выбери загрузку** с CD/DVD
3. **Подожди загрузки** (2-3 минуты)
4. **Автоматический вход** как пользователь `neet`
5. **Hyprland запустится** автоматически

## Решение проблем

### ❌ "archiso not found"
```bash
sudo pacman -S archiso
```

### ❌ "Permission denied"
```bash
chmod +x build.sh
sudo ./build.sh
```

### ❌ "No space left"
```bash
# Освободи место
sudo pacman -Sc
sudo rm -rf work/ out/
df -h .
```

### ❌ Чёрный экран при загрузке
- Увеличь ОЗУ VM до 8GB
- Включи 3D-ускорение
- Подожди 2-3 минуты
- Нажми `Ctrl+Alt+F2` для TTY

**Полное руководство:** [TROUBLESHOOTING.md](TROUBLESHOOTING.md)
