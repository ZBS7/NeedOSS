================================================================================
                    СБОРКА NEETOS ЧЕРЕЗ GITHUB ACTIONS
                  Самый простой способ без установки Linux!
================================================================================

🎯 ЧТО ЭТО ДАСТ:

  ✅ Сборка ISO в облаке (бесплатно)
  ✅ Не нужен Linux на твоём ПК
  ✅ Не нужна VirtualBox с Manjaro
  ✅ Просто нажми кнопку и жди
  ✅ Скачаешь готовый ISO за 30-40 минут


================================================================================
⚡ БЫСТРАЯ ИНСТРУКЦИЯ (5 ШАГОВ)
================================================================================

ШАГ 1: РЕГИСТРАЦИЯ НА GITHUB
────────────────────────────────────────────────────────────────────────────
  1. Иди на https://github.com/signup
  2. Введи email, пароль, username
  3. Подтверди email
  4. Готово!

  Уже есть аккаунт? Переходи к Шагу 2


ШАГ 2: СОЗДАЙ РЕПОЗИТОРИЙ
────────────────────────────────────────────────────────────────────────────
  1. На GitHub нажми "+" → "New repository"
  2. Заполни:
     - Repository name: neetos
     - Description: NeetOs Custom Linux Distribution
     - ☑ Public
  3. Нажми "Create repository"


ШАГ 3: ЗАГРУЗИ ФАЙЛЫ
────────────────────────────────────────────────────────────────────────────
  1. В репозитории нажми "Add file" → "Upload files"
  
  2. Перетащи ВСЕ файлы из папки neetos:
     ✓ Все .md файлы (документация)
     ✓ Все .sh файлы (скрипты)
     ✓ Все .txt файлы
     ✓ profiledef.sh
     ✓ packages.x86_64
     ✓ pacman.conf
     ✓ Папку .github/    ← ВАЖНО!
     ✓ Папку configs/
     ✓ Папку airootfs/
  
  3. В commit message напиши: "Initial commit"
  
  4. Нажми "Commit changes"

  ⚠️ ВАЖНО: Папка .github должна загрузиться!
            Она начинается с точки - её может быть не видно.
            В Windows включи показ скрытых файлов.


ШАГ 4: ЗАПУСТИ СБОРКУ
────────────────────────────────────────────────────────────────────────────
  1. Перейди на вкладку "Actions" (вверху)
  
  2. Если появится запрос - нажми "I understand, enable them"
  
  3. Слева выбери "Build NeetOs ISO"
  
  4. Справа нажми "Run workflow"
  
  5. Выбери ветку "main" и нажми зелёную "Run workflow"
  
  6. Сборка началась! ⏳


ШАГ 5: СКАЧАЙ ISO
────────────────────────────────────────────────────────────────────────────
  Подожди 30-40 минут пока статус станет зелёным ✅
  
  1. Нажми на завершённый workflow (зелёная галочка)
  
  2. Прокрути вниз до "Artifacts"
  
  3. Скачай "neetos-iso" (ZIP-архив)
  
  4. Распакуй ZIP → получишь neetos-*.iso
  
  5. Запусти в VirtualBox!
  
  🎉 ГОТОВО!


================================================================================
🎮 ЗАПУСК В VIRTUALBOX
================================================================================

Теперь у тебя есть ISO! Создай VM:

  1. VirtualBox → Создать
  2. Тип: Linux → Arch Linux (64-bit)
  3. ОЗУ: 4096 MB
  4. Диск: 20 GB
  5. Настройки → Дисплей → Видео 128MB, 3D включено
  6. Настройки → Носители → Подключи neetos-*.iso
  7. Запусти!
  8. Логин: neet / neet


================================================================================
🔧 ЕСЛИ НУЖНО ИЗМЕНИТЬ И ПЕРЕСОБРАТЬ
================================================================================

  1. На GitHub открой нужный файл (например packages.x86_64)
  2. Нажми карандаш (Edit)
  3. Внеси изменения
  4. "Commit changes"
  5. Перейди в Actions → Run workflow снова
  6. Получи новый ISO!

================================================================================
❓ ЧАСТЫЕ ПРОБЛЕМЫ
================================================================================

❌ "Workflow не появился в Actions"
   → Проверь что папка .github/workflows/ загрузилась
   → В Windows включи показ скрытых файлов
   → Перезагрузи страницу GitHub

❌ "Сборка упала с ошибкой"
   → Открой failed workflow
   → Нажми на job "build"
   → Разверни failed step
   → Покажи мне ошибку

❌ "Не могу скачать ISO"
   → Artifact хранится 7 дней
   → Если удалился - просто пересобери
   → Run workflow снова

❌ "Превысил лимит GitHub Actions"
   → Бесплатно: 2000 минут/месяц
   → Одна сборка ~30-40 минут
   → Хватит на ~50 сборок
   → Подожди до следующего месяца
